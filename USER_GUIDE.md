# 音檔轉 OGG 工具使用教學

這個工具可以批量把 `.mp3` / `.wav` 轉成 `.ogg`，並支援前綴命名、順序命名、轉換後自動整理原始檔。

## 1. 開啟工具

請執行：

```text
音檔轉OGG.exe
```

如果你拿到的是壓縮檔，請先解壓縮後再執行 `音檔轉OGG.exe`。

## 2. 第一次使用：下載轉檔程式

這個工具需要 `ffmpeg.exe` 來實際轉換音檔。

第一次開啟時，如果畫面下方顯示找不到轉檔程式：

1. 按「下載轉檔程式」
2. 等待下載進度條跑完
3. 工具會自動把 `ffmpeg.exe` 放到正確位置

下載檔案較大，等待數十秒到數分鐘都是正常的，依網路速度而定。

## 3. 選擇資料夾

畫面上有三個主要路徑欄位：

- `輸入資料夾`：放 `.mp3` / `.wav` 的資料夾
- `輸出資料夾`：轉換後 `.ogg` 要存放的位置
- `轉檔程式`：`ffmpeg.exe` 的位置，通常下載後會自動填入

選好輸入資料夾後，按「重新掃描」更新清單。

## 4. 勾選要轉換的檔案

清單左側的「轉換」欄位可以勾選檔案。

- 有勾選：會轉換
- 沒勾選：不會改名，也不會轉換

## 5. 一般命名：套用檔名前綴

如果想在輸出檔名前面加文字：

1. 在「檔名前綴」輸入文字，例如 `sfx_`
2. 按「套用命名」

範例：

```text
大砲.mp3 -> sfx_大砲.ogg
受傷.mp3 -> sfx_受傷.ogg
```

## 6. 順序命名

如果想要輸出成連號檔名：

1. 在「檔名前綴」輸入文字，例如 `sci-fi`
2. 勾選「按照順序命名」
3. 在「起始編號」輸入 `001`
4. 按「套用命名」

範例：

```text
sci-fi_001.ogg
sci-fi_002.ogg
sci-fi_003.ogg
```

編號位數會依照起始編號保留。

例如：

- 起始編號 `001` -> `001`, `002`, `003`
- 起始編號 `010` -> `010`, `011`, `012`
- 起始編號 `1` -> `1`, `2`, `3`

未勾選的檔案不會參與順序命名。

## 7. 覆蓋舊檔

如果輸出資料夾已經有同名 `.ogg`：

- 沒勾「覆蓋舊檔」：工具會略過該檔案
- 有勾「覆蓋舊檔」：工具會覆蓋舊檔

一般建議先不要勾，確認輸出沒問題後再視情況使用。

## 8. 開始轉換

確認清單和輸出檔名後，按「開始轉換」。

轉換過程中：

- 進度條會顯示進度
- 下方紀錄區會顯示目前轉換狀態
- 成功、略過、失敗數量會在完成後顯示

## 9. 轉換完成後檔案在哪

轉換成功後：

- `.ogg` 會出現在「輸出資料夾」
- 原始 `.mp3` / `.wav` 會被移到輸入資料夾底下的 `converted` 資料夾

範例：

```text
輸入資料夾\大砲.mp3
```

轉換成功後會變成：

```text
輸出資料夾\大砲.ogg
輸入資料夾\converted\大砲.mp3
```

如果 `converted` 裡已有同名檔，工具會自動加 `_1`, `_2`，避免覆蓋。

## 10. 切換淺色 / 深色模式

右上角有主題切換按鈕：

- `☾`：切換到深色模式
- `☀`：切換到淺色模式

## 11. 常見問題

### 按「下載轉檔程式」跑很久正常嗎？

正常。`ffmpeg` 檔案較大，下載和解壓縮都需要時間。

### 清單沒有看到音檔怎麼辦？

請確認：

- `輸入資料夾` 是否正確
- 檔案格式是否為 `.mp3` 或 `.wav`
- 是否需要按「重新掃描」

### 為什麼有些檔案沒有轉換？

請確認該檔案左側「轉換」欄位是否有勾選。

### 為什麼輸出檔案被略過？

通常是輸出資料夾已有同名 `.ogg`，且沒有勾選「覆蓋舊檔」。

### 可以手動改輸出檔名嗎？

可以。直接在清單的「輸出 .ogg 檔名」欄位修改即可。
